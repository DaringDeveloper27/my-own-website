# Personal Portfolio Website 

Welcome to the repository for my personal portfolio website! This project is a responsive, multi-page showcase of my software engineering work, robotics projects, and my journey as a self-taught developer. 

Built as part of my learning path through **Harvard's CS50x**, the site features a sleek, Apple-inspired premium aesthetic utilizing custom styling over a clean structural layout.

---

## Built With

* **HTML5** – Structured multi-page architecture (`index.html`, `about.html`, `projects.html`, `contact.html`).
* **CSS3** – Custom global styling featuring an Apple-themed color palette, responsive design layout refinements, and interactive link hover transitions.
* **Bootstrap 5** – Leveraged for its robust grid system, utility classes, and fully responsive layout structures.
* **JavaScript** – Integrated via CDN for interactive components and responsive behaviors.

---

## Features Included

* **About Me Narrative:** A deep look into my background starting with robotics at age 10, transitioning into advanced C++ software engineering, and studying core computer science algorithms.
* **Interactive Skill Badges:** An aligned technologies grid matching official brand-identity colors for languages like C++, Python, Java, C, and tools like Git/GitHub.
* **Responsive Layout:** Engineered to dynamically adapt and scale elegantly across mobile devices, tablets, and desktop displays.
* **Optimized Architecture:** Consolidated stylesheets and clean document structures built to professional standards.
