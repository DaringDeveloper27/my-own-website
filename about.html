My Story

I was introduced to programming at the age of 10 when my father put me in a course about Robotics. This course covered development of simple circuits with Arduino Uno and C++. From there, I have fallen deep in the world of Robotics and Computer Science. While building these projects, I learned more on Electrical Components such as the Ultrasonic Sensors or Buzzers, how they work and how they can in help to us. Another aspect which I enjoyed was the Coding. Despite being a tough language for me, C++ is my favorite to code in after I realized how powerful it is. From there, I went to learn the Development of Software with C++. The Basics were pretty easy for me to catch due to my prior experience in Robotics. After learning sufficient amount of knowledge, I went on to build my first project which was a Calculator built in C++. During summer of 2023, I enrolled in a Data Science Course. We learned Python which became an interest for me during this time. Hence, I decided to learn more about it by joining 100 days of Code with Python by Dr. Angela Yu. My third language was Java which I learned though Abdul Bari, an instructor in Udemy. A major part which has made me obsessed with Programming was enrolling in Harvard's CS50x Course on EDX during the summer of 2025. From this course, I learned C, Python, HTML, CSS, and a little bit of JavaScript and SQL. I also made myself busy with Advanced Concepts such as Memory Management, Data Structures and Algorithms. Now, I decided to tackle Data Structures and Algorithms using C++ such that I am able to solve questions in LeetCode.
